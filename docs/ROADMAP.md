# Roadmap

Future enhancements and ideas:

- Zero-knowledge proofs for private verification
- Multi-signature or role-based access control
- File versioning and audit logs
- Integration with IPFS pinning services
- Mobile-friendly user interface
- Automatic re-encryption on key rotation
- Support for larger files via chunking
- Performance improvements on backend hashing
- Dockerization and cloud deployment scripts

The roadmap is intentionally flexible; contributions and customizations are encouraged.
